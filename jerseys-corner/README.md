# Jersey's Corner

Family-owned restaurant in Parowan, UT — on Main Street since 1958.
Static HTML/CSS, no build step. Open `index.html` to view locally.

## Pages
- `index.html` — home
- `menu.html` — full menu
- `about.html` — our story
- `contact.html` — find us

## Stack
- Vanilla HTML/CSS, single shared stylesheet
- Google Fonts: Playfair Display + DM Sans
- Tiny inline JS on home page that highlights today's hours

## Deployment
Push to GitHub, connect to Netlify. No build settings required.
